Inline Four Diesel Engine Project

This was a final project for my EML 2023 Computer Aided Graphics and Design class. This class taught me how to use Solidworks and provided me with the opportunity to get certified at the end of the Semester. For this class we had a semester long final project where we had to showcast a product, machine, or mechanism that integrates as many course-topic as possible. For this project I worked with one other person. We chose to showcast a diesel engine. We then designed, assembled, and animated a inline four diesel engine.
